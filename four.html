Create a New Object Class

This creates a class which defines six-sided dice and a method which generates a roll:

class Die

  def roll

    1 + rand(6)

  end

end

This populates a new array with new objects of sixed-sided dice:

dice = [Die.new, Die.new, Die.new]

This iterates through the array, calling the Die class's roll method:

dice.each do |die|

  puts die.roll

end

Note how the method call here is to the array iterator, hence the lower-case die.
