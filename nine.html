Ruby Notes 9

Code Blocks

- any code between do and end, or between braces: {}

Associate a Block with a Method Call

The yeild statement produces one code block:

def callBlock

  yield

end


callBlock { puts "content" }

pass params through a block call

def callBlock

  yield

end


callBlock { p1|,p2|,p3 }

Yes, it's just that simple.

Pass a Block through a Predefined Method Call

Remember a.each?

a.each { |elm| puts elm }

You can substitute braces for the do and end statements if you like, you javaScripter,you.
