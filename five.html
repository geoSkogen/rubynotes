Create an Instance Variable - Object Element

Where method variables last only through the method call, an object's instance variables last as long as the object.

class Die

  def roll

    @faceNumber = 1 + rand(6)

  end

  def face

    @faceNumber

  end

end


die = Die.new  #assigns new object to reference variable

die.roll

puts die.face

The use of the instance variable allows us to store the results of the roll and inspect it without rolling the die. However, if we attempt to inspect the roll value before the die has been rolled, Ruby returns a nil.

The Initialize Method

Initialize is always called when a new class object is created. Insert the this method after the class declaration:

class Die

  def initialize

    roll

  end

  def roll

    @faceNumber = 1 + rand(6)

  end

  def face

    @faceNumber

  end

end

Now any die object created with the Die class will have a face value, even before we roll it - just like an actual die.
