Ruby Notes 2

Branching - Conditional Statements

Code blocks for conditions aren't enclosed in brackets, or anything. Just keep the conditionals on their own separate lines and close the whole thing with an end.

if   

  #code block

else   

  #code block

end

while   

  #code block

end   

True, false, and, or, and not are just that: true    false    and    or    not   

Hurray! Array Methods

The .each iterator:

a.each do |am|

#code block using am

end

For each array member, Ruby executes any lines of code which follow, wherein am represents the current array member. Identify iterative parameter between pipes and use it in the code block that follows. As always, close with end statement.

A Few More Common Methods

puts a.to_s   displays ["m1", "m2","m3"]

puts a.join(",")   displays m1, m2,m3

a.push "newElement"   add new element in last position

a.pop   removes last element and displays it

a.last   displays last element only

a.sort   alphabetically re-orders array
