Class Variables

If we want to track which songs get played the most, we use an instance variable; each song records its number of plays - each object has a number of plays variable. But if we want the jukebox to count the total number of songs it's played, we need a class variable

Declare the class variables proir to the initialize method, declare the instance variables with the initialize method.

class Song

@@plays = 0

  def initialize(name, artist, duration, plays)

    @name = name

    @artist = artist

    @duration = druation

    @plays = 0

  

  def play

    @plays +=1

    @@plays +=1

    "song: #{@plays} total, jukebox: #{@@plays} total"

  end

end


Now, create a couple different Song objects:

s1 = Song.new("Panasonic Youth","DEP","2:33")

s2 = Song.new("Phthisis","SGM","3:46")

If our first "play is s1.play, the program will return song: 1 total, jukebox: 1 total. Following with s2.play, the program returns song: 1 total, jukebox: 2 total, etc.
