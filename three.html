A Simple Logic Gate

binary = false

while (not binary)

  puts "Enter either Yin or Yang:"

  goodInput = gets.chomp.downcase


  if (goodInput == "Yin" or goodInput == "Yang")

    binary = true

  else

    puts = "Enter Yin or Yang only."

  end    #closes if else statements

end  #closes while loop

Create a condition, then invoke a loop with a pertaining conditional statement. Note that there is no keyword for defining variables or for assignment.

How to Write a Simple Global Method

def sayHand

  puts "Hand"

end


Add Paramaters to Your Method

def sayHand doHand

  puts "Hand" * doHand

end

Now your method takes a numeric parameter. By typing sayHand 3, for example the method will take the 3 in place of doHand.

Methods May Contain Local Variables

def squareIt val

  valSquared = val * val

  puts val.to_s + " squared is " + valSquared.to_s

end
